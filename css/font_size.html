01. font-size 단위의 고찰

font-size는 문자의 글자 크기를 지정하는 속성입니다.
글자 크기는 명시하지 않을 경우 16px이 기본값입니다.

단위 설명
px 해상도에 따라 상대적으로 다라지는 기본 단위
% 부모 요소의 글자 크기를 100% 기준으로 계산한 % 단위
em 부모 요소의 글자 크기를 100% 기준으로 계산한 100 분의1 단위
rem rem은 em과 비슷하나 부모 요소가 아닌 최상위 요소의 크기를 100% 기준으로 계산
rem은 r은 root 즉, html 요소 / html요소에서 지정한 글자 크기가 1rem이 됨.
vw, vh vw는 뷰포트 너비값의 100분의 1 단위, vh는 뷰트 높이값의 100분의 1 단위
