HTML 기본 문서

HTML 기본 문서의 <html> 요소는 <head> 요소와 <body> 요소로 구성되어 있으며, head 요소 안에는 meta, title, link, style, script 등의 요소들이 올 수 있습니다.

요소명 설명
<meta> HTML 문서의 설명, 키워드, 문서의 작성자 등을 지정
<title> Html 문서 제목을 지정
<link> 외부의 CSS파일을 HTML 문서에 연결할 때 사용
<style> HTML 문서 내에서 CSS (스타일) 정의
<script> 자바스크립트를 HTML 문서 내부에서 정의하거나 외부에서 자바스크립트 파일을 불러올 때 사용

속성 = <html lang="ko">나 <meta charset="UTF-8">처럼 태그 내부에 부가적인 설정값을 선언하는 것

lang 속성 = HTML 문서에서 사용하는 언어/lang = "ko"는 한국어, lang="en"은 영어 lang의 중요성 : 스크린 리더(음성낭독 프로그램)에서 언어 설정에 따라 다르게 읽어 주기 때문에 HTML문서에서 꼭 정의해야 하는 속성

<meta> 태그의 charset 속성의 값 UTF-8을 '문자 인코딩(charset)' 방식이라고 하는데 UTF-8의 경우 겅의 대분분의 문자를 처리할 수 있기 때문에 HTML의 문자 인코딩 방식으로 주로 사용되고 있음.
<meta> 태그에서 알아 두어야 할 속성 중 name 속성과 content 속성이 있음-> CSS파트에서 반응형 웹 기술을 사용하기 위한 속성임.

<meta name = "viewport" />

뷰포트라고 하면 문서를 조절해 주는 속성이다.
